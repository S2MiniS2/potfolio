# 🧑‍💻 Portfolio

프론트엔드 개발자 박민희의 포트폴리오입니다.  
사용자 경험과 비즈니스 로직을 함께 고려하는 개발을 지향합니다.

---

## 📌 About

이 프로젝트는 개인 포트폴리오 사이트로,  
프로젝트 경험과 기술 스택, 그리고 개발 방향성을 효과적으로 전달하기 위해 제작되었습니다.

---

## 🚀 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Styled-components
- Figma

---

## ✨ Features

- 반응형 UI (모바일 / 태블릿 / 데스크탑 대응)
- 프로젝트 중심 구조 설계
- 사용자 흐름을 고려한 UI/UX 구성
- 컴포넌트 기반 설계
- 유지보수를 고려한 코드 구조

---

## 📂 Project Structure

app/  
components/  
styles/  
public/  

- app/ : 페이지 및 라우팅  
- components/ : 재사용 가능한 UI 컴포넌트  
- styles/ : 글로벌 스타일 및 테마  
- public/ : 이미지 및 정적 파일  

---

## 🧠 What I Focused On

- 단순 구현이 아닌 **비즈니스 로직에 맞는 구조 설계**
- 컴포넌트 단위 분리 및 재사용성
- 유지보수 가능한 코드 작성
- 사용자 입장에서의 UX 흐름 설계
